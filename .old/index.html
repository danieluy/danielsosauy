Used technologies

HTML 5

Page structuring and articulation of the other technologies involved.

Javascript

This technology has a double role. In the user interface it makes possible the interaction of the users with the page, on the other hand it is in charge of running the server that serves the page.

CSS3

Defines and applies the styles and animations that enrich the interaction of users with the site.

React

A Javascript library for building user interfaces.

Node.js

Runtime used on the server of this page. This is the technology that allows JavaScript to be used as a server side language.

Express

Facilitates server-side infrastructure management in Node.js.

Webpack

Bundling and minification of JavaScript and CSS files as well as other resources, to optimize the resource loading. In the case of this page it is used from the React scripts.
